## Завдання для попередньої підготовки

### Glossary of Basic English Terms
CLI (Command Line Interface) — a text-based interface that allows the user to enter commands to control the operating system.  
Terminal — a program that provides access to the command line in a graphical Linux environment.  
Virtual Terminal — a text-based Linux console that works independently of the graphical interface and is used for logging into the system and executing commands.  
Kernel — the core of the operating system that manages memory, CPU, devices, and the execution of applications.  
Application — a software program that performs specific tasks and uses system resources provided by the kernel.  
Process — a running instance of a program that is managed and controlled by the operating system kernel.  
GUI (Graphical User Interface) — a graphical user interface that allows users to interact with the system using windows, menus, and buttons.

## Визначення основних понять
CLI-режим (Command Line Interface) — це режим роботи операційної системи, у якому користувач взаємодіє з системою шляхом введення текстових команд з клавіатури.  

Термінал на основі графічного інтерфейсу користувача — це програма, яка працює у графічному середовищі та надає користувачу доступ до командного рядка для виконання команд.  

Віртуальний термінал — це текстова консоль операційної системи Linux, яка працює незалежно від графічного інтерфейсу та використовується для входу в систему і виконання команд.

## Хід роботи

### Робота в графічному режимі в ОС сімейства Linux (робота з інтернет-джерелами)
Я обрала GNOME.

#### Основне меню
У графічній оболонці GNOME основне меню реалізоване у вигляді режиму Activities. Воно відкривається натисканням клавіші Super або переміщенням курсора у верхній лівий кут екрана. В режимі Activities користувач може переглядати відкриті вікна, запускати програми та керувати робочими столами. На відміну від класичного меню «Пуск», у GNOME всі ці функції об’єднані в одному інтерфейсі.

#### Панелі швидкого доступу
Панель швидкого доступу (Dock) розташована з лівого боку екрана. На ній відображаються значки найчастіше використовуваних програм, таких як веббраузер, файловий менеджер або термінал. За допомогою цієї панелі можна швидко запускати програми, а також перемикатися між вже відкритими додатками.

#### Пошук
У GNOME передбачена зручна функція глобального пошуку, яка доступна в режимі Activities. Пошук дозволяє швидко знаходити програми, файли та системні налаштування шляхом введення ключових слів. Це значно спрощує роботу з системою, оскільки немає необхідності вручну переглядати список встановлених програм.

#### Доступ до нових робочих столів (Workspaces)
Графічна оболонка GNOME підтримує роботу з кількома робочими столами (workspaces). У режимі Activities користувач може створювати нові робочі столи та перемикатися між ними. Це дозволяє зручно організовувати робочий простір, розміщуючи різні програми на окремих робочих столах і зменшуючи безлад на екрані.

### Зміна користувача на root
У графічному середовищі GNOME для переходу до адміністративних прав зазвичай використовується термінал із командою `sudo`, а не прямий вхід у root через меню. Для отримання прав root відкривається Terminal і виконується команда `sudo -i` (після введення пароля користувача). Це дозволяє виконувати адміністративні дії.

## Робота в середовищі мобільної ОС (iOS, iPhone 17 Pro)

### Опишіть головне меню вашої мобільної ОС, який графічний інтерфейс вона використовує?
Головне меню мобільної операційної системи iOS на iPhone 17 Pro представлене у вигляді домашнього екрана (Home Screen), на якому розміщені значки встановлених додатків і віджети. Користувач може перегортати сторінки головного екрана, об’єднувати програми у папки та змінювати їх розташування.  
Операційна система iOS використовує сенсорний графічний інтерфейс, що базується на керуванні за допомогою дотиків, жестів і анімацій, завдяки чому робота з пристроєм є зручною та інтуїтивно зрозумілою.

### Опишіть меню налаштувань компонентів мобільного телефону
Меню налаштувань iPhone 17 Pro зосереджене в системному додатку «Налаштування». У ньому користувач може керувати основними компонентами пристрою: мережевими підключеннями (Wi-Fi, Bluetooth, мобільний зв’язок), параметрами екрана, звуку, сповіщень, безпеки та конфіденційності. Також доступні налаштування облікового запису Apple ID, керування додатками, оновлення операційної системи та системні параметри пристрою.

### Використання комбінацій клавіш для виконання спеціальних дій
В iPhone 17 Pro передбачено використання апаратних комбінацій кнопок для швидкого виконання спеціальних дій. За допомогою поєднання кнопок можна зробити знімок екрана, викликати меню вимкнення пристрою або виконати примусове перезавантаження. Використання таких комбінацій дозволяє оперативно керувати пристроєм без необхідності переходу до меню налаштувань.

### Вхід у систему та завершення роботи пристрою. Особливості налаштувань живлення батареї
Вхід у систему на iPhone 17 Pro здійснюється за допомогою біометричної автентифікації Face ID або введення коду доступу. Завершення роботи пристрою виконується через системне меню вимкнення або налаштування операційної системи.  
У налаштуваннях живлення користувач може переглядати інформацію про рівень заряду акумулятора, статистику споживання енергії додатками та активувати режим енергозбереження. Режим енергозбереження дозволяє зменшити фонову активність системи та продовжити час автономної роботи пристрою.

## Контрольні запитання

1) Наведіть приклади серверних додатків Linux для сервера баз даних, серверів розсилки повідомлень та файлообмінників.  
У Linux існує велика кількість серверних додатків для різних завдань. Для серверів баз даних широко використовуються MySQL, MariaDB та PostgreSQL, які забезпечують зберігання та обробку великих обсягів даних. Для серверів розсилки повідомлень застосовуються такі програми, як Postfix, Sendmail та Exim, які відповідають за обробку та доставку електронної пошти. Для організації файлообміну використовуються Samba, FTP-сервери (vsftpd) та NFS, які дозволяють надавати доступ до файлів у мережі.

2) Порівняйте оболонки Bourne, C, Bourne Again (Bash), tcsh, Korn shell (ksh) та zsh.  
Bourne shell є однією з перших оболонок Unix і має базовий функціонал. C shell орієнтована на синтаксис мови C та зручна для інтерактивної роботи. Bourne Again shell (Bash) є найпоширенішою оболонкою в Linux та поєднує можливості Bourne shell із додатковими зручностями, такими як історія команд і автодоповнення. tcsh є вдосконаленою версією C shell. Korn shell поєднує можливості sh і csh та часто використовується в серверних і корпоративних середовищах. zsh є сучасною оболонкою з розширеним автодоповненням і широкими можливостями налаштування.

3) Для чого потрібен менеджер пакетів. Які менеджери пакетів ви знаєте у Linux?  
Менеджер пакетів використовується для встановлення, оновлення та видалення програмного забезпечення в операційній системі Linux. Він автоматично керує залежностями між програмами та спрощує адміністрування системи. До відомих менеджерів пакетів належать apt, dnf, yum, pacman та zypper.

4) Які засоби безпеки використовуються в Linux?  
У Linux використовується система розмежування прав доступу до файлів і каталогів, облікові записи користувачів та обмежене використання прав адміністратора. Для підвищення безпеки застосовуються механізми SELinux або AppArmor, мережеві екрани, а також регулярні оновлення системи.

5) Чому використання віртуалізації стало таким актуальним?  
Віртуалізація дозволяє ефективніше використовувати апаратні ресурси комп’ютера шляхом запуску кількох операційних систем на одному фізичному пристрої. Вона широко застосовується для тестування програм, навчання та зменшення витрат на обладнання.

6) Як ви розумієте поняття контейнеризації?  
Контейнеризація — це технологія запуску програм у ізольованих середовищах, які використовують одне ядро операційної системи. Контейнери містять усі необхідні залежності для роботи програм, що забезпечує однакову поведінку додатків у різних середовищах.

7) Які переваги та недоліки використання програмного забезпечення з відкритим кодом?  
До переваг програмного забезпечення з відкритим кодом належать безкоштовність, прозорість, можливість модифікації та активна підтримка спільноти. До недоліків можна віднести відсутність офіційної підтримки у деяких проєктах та складність налаштування для недосвідчених користувачів.

8) Скільки активних віртуальних консолей може бути в Linux за замовчуванням і як між ними перемикатися?  
У Linux за замовчуванням доступно шість текстових віртуальних консолей. Перемикання між ними здійснюється за допомогою комбінацій клавіш Ctrl + Alt + F1 – F6.

9) Яка віртуальна консоль виконує функцію графічної оболонки?  
Графічна оболонка в Linux зазвичай працює на віртуальній консолі tty7 або, у деяких дистрибутивах, tty1.

10) Чи можлива реєстрація в системі Linux декілька разів під одним і тим же системним ім’ям? Які переваги це може надати?  
Так, у Linux можливий одночасний вхід у систему під одним і тим самим обліковим записом з різних терміналів або консолей. Це дозволяє виконувати кілька завдань паралельно та підвищує зручність роботи користувача.
